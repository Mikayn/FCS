## Foundation Assignment – Secure Encoding, Computational Complexity, and Database Normalization

This repository contains academic project demonstrating the **practical role of encoding formats in secure data exchange**, **computational complexity classes (P vs NP)**, and  **database normalization techniques (1NF, 2NF, 3NF)**. The work is implemented using **Python, Docker, and SQL**, and is designed to reflect realistic computing scenarios such as secure communication, algorithmic problem-solving, and relational database management.

---

## Project Overview

The assignment is divided into **three main tasks**, each focusing on a core concept in computer science and cybersecurity:

### **Task 1 – Secure Data Exchange**
This task examines widely used encoding formats such as **Base64, ASCII, Quoted Printable**, explaining how they are used during secure data transmission. These encodings are demonstrated in combination with communication protocols and technologies including **HTTPS, TLS, and SMTP**. A Docker-based environment is used to simulate secure file transfers in HTTPS, supported by diagrams, execution outputs, and a security analysis that evaluates their purpose, limitations, and relevance in real systems.

### **Task 2 – Computational Complexity (P vs NP)**
This task uses a **classroom seating arrangement problem** to clearly show the difference between **P** Problems and **NP** Problems. A **brute force algorithm** is compared with a **heuristic approach** to show how optimization impacts performance. The task includes Python implementations, runtime outputs, and a written analysis that connects theoretical complexity concepts with observed behavior.

### **Task 3 – SQL and Database Normalization**
The final task focuses on designing a **college club membership management database**. Starting from an initial unnormalized table, the database is refined into **First Normal Form (1NF)**, **Second Normal Form (2NF)**, and **Third Normal Form (3NF)**. This section includes an ER diagram, SQL schema definitions, data insertion scripts, JOIN queries, along with a detailed normalization explanation.
